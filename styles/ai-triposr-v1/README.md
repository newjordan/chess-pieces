# ai-triposr-v1

AI-generated chess style set.

- Module ID: `ai-triposr-v1`
- Author: Frosty40
- License: CC-BY 4.0
- Format: GLB
- Piece types: king, queen, rook, bishop, knight, pawn
- Sides: white + black (currently same geometry, intended to be differentiated with materials in-engine)
- Pipeline: SDXL image generation (ComfyUI) -> TripoSR image-to-3D conversion
- Poly stats: see `mesh_stats.json`
- Machine-readable manifest: `style.json`
