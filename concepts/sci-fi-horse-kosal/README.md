# Sci-Fi Horse Kosal Concept

Single-piece SVG concept extracted from the sci-fi horse exploration.

This is intentionally stored under `concepts/` instead of `styles/` because it is not a complete 12-piece GLB style module.

## Files

- `horse-mono.svg`: clean one-color `currentColor` SVG.
- `horse-lineart.svg`: line-art `currentColor` SVG for UI/mockup use.
- `source.json`: source and experiment metadata.

## Source

- Source project: `chess_svg_library`
- Source asset: `library/final_12/knight/white.svg`
- Provider: `deverac-piece-packager`
- Set: `kosal`
- License: `GPL-2.0`

The 30 sci-fi horse concepts were shader treatments over this silhouette. This folder preserves the base horse shape without the procedural color/effects.
